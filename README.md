# Resistor Colour Code Calculator

A web-based **Resistor Colour Code Calculator** built using **HTML, CSS, and JavaScript**.  
This tool helps calculate resistance values and tolerance ranges for **4-band and 5-band resistors** based on standard electronics colour coding.
